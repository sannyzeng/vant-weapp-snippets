### Snippets List
1. 基于[Vant Weapp](https://youzan.github.io/vant-weapp/#/home) v1.9.1， 支持所有 Vant Weapp 标签。
2. 支持在 app.json/page.json `usingComponents` 中导入模块。
3. 支持 wxss 快速导入公共样式。
4. 支持 js 中快速导入以及使用 Toast、Dialog、Notify 等模块。
5. 支持自动补全
